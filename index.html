GNU Linux - De Onde Veio? Para Onde Vai?

Se você chegou a esse site então é bem provável que você esteja dando seus primeiros passos nesse sistema operacional e ainda deve estar meio perdido. Então deixa eu explicar brevemente o histórico desse projeto.

Primeiramente tivemos o surgimento de um projeto de sistema operacional encabeçado por Richard Stallman, em 1984, chamado GNU com o objetivo de fazer um sistema operacional formado apenas por software livre. Tal sistema deveria ser compatível com o já existente UNIX, mas sem utilizar qualquer código fonte proveniente deste. Vários desenvolvedores abraçam a causa de Stallman e passam a desenvolver as principais ferramentas para o projeto, como um compilador de linguagem C, editores de texto e outras ferramentas.

Em 1992 o ecossistema do sistema operacional GNU estava quase completo. Faltava apenas o núcleo(kernel) do sistema. Stallman e sua equipe vinham trabalhando no núcleo chamado de HURD. Porém também em 1992, um jovem finlandês resolve mudar a licença do seu núcleo para uma licença de software livre, e tal núcleo era capaz de trabalhar em perfeita harmonia com todas as ferramentas do projeto GNU. Isso permitiu que Stallman e sua equipe lançassem a primeira versão de um sistema operacional livre. Tal jovem era Linus Torvalds e o seu núcleo ficou conhecido pelo que chamamos de Linux.

Desde então o projeto vem crescido, mais e mais pessoas vem contribuindo para a base de código do núcleo Linux e do projeto GNU. Muitas empresas importantes como Intel, IBM, AMD, Lenovo, Motorola, Google, Nokia e a própria Microsoft já fizeram importantes contribuições para a base desses softwares. Hoje é possível encontrar o GNU/Linux operando nos mais diversos tipos de dispositivos, desde celulares e roteadores até grandes servidores, há também iniciativas de usando-o em satélites.
